# carbon-fiber-Filament-Winder
A carbon fiber filament winder to make tubes  (uses aluminium profile channels)
im using raspberyy pi pcio as the controller for this project
dimension of the winder : 1000mm x 250mmx350mm

max dimension of the tube which we can make : 50mm radius 700mm length

you can find my cad file here https://cad.onshape.com/documents/22d46236cee3903220293ec8/w/06b310adcdbfb510975e655e/e/3099af29fedbc7c387d2577e
